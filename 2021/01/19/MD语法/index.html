插入表情

  • &#xCODE;

CODE 可以从 Emoji Unicode Tables 中查到。例如查到了 😀 表情对应的 Unicode 编码为 U+1F600，则与此表情对应的 CODE 为 1F600 (舍弃前面的 U+)。我们只需在 Markdown 文档中输入 ·&#x1F600; 即可显示为 😀 。

  • :CODE:

查找：EMOJI CHEAT SHEET ，例如:expressionless: 显示为 😑。

然后我发现在 Typora 中键入:以及某字母，就会弹出表情选项😂

第二种方法部署后网页并不能显示出表情，需要安装 hexo-filter-github-emojis 插件：

npm install hexo-filter-github-emojis --save

然后在 _config.yml 增加配置：

githubEmojis:
  enable: true
  className: github-emoji
  unicode: true
  styles:
    display: inline
    vertical-align: middle 
  localEmojis:

详见 hexo-filter-github-emojis
